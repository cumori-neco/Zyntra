# Zyntra
A simple rhythm game engine built for Unity that nobody asked for.
* * *
## 🏃 Current Progress
* ✅ Pretty much usable.
* ✅ Logic & Basic functions ready.
* ❌ Data & Scoring not finished.
* ❌ Looks messy as fuck.
* ❌ Still requires more testing.

## 🎯 Goals
I wish people can make rhythm games more easily, so I made this stuff. With Zyntra, you only need to focus on the game logic and visuals, while Zyntra handles audio, objects, level data, judgements and scoring for you.

* * *
## 🔨 Development
This library must be used within Unity with version 2022.3 and newer.

That's it and I don't expect this to be found by people.
